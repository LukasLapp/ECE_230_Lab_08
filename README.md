## Lab Questions

### In plain English describe the function and use of a multiplexer.
A multiplexer is a component that selects one input, from several possible inputs, and sends it to a single output. For example, if a multiplexer has four inputs, it can pick one of those four signals and send it to the output depending on the value of the select bits

### In plain English describe the function and use of a demultiplexer.
A demultiplexer is a component that takes one input signal and sends it to one of the several possible outputs.  This device receives a signal and, based on the selector, it decides which output should receive the signal.

### What other uses might these circuits have? (Think Shannon’s)
Multiplexers and Demultiplexers can also be used to implement Boolean logic functions.
